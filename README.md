# Cyber Security Internship Project

## Project Title
Strengthening Security Measures for a Web Application

---

## Project Overview

This project was completed as part of a Cyber Security Internship task. The objective was to analyze a vulnerable web application, identify common vulnerabilities, and implement security improvements using secure coding practices and cybersecurity tools.

The OWASP Juice Shop application was used for vulnerability assessment and penetration testing.

---

## Technologies and Tools Used

- Node.js
- Express.js
- OWASP Juice Shop
- OWASP ZAP
- Nmap
- VS Code
- Thunder Client
- JWT Authentication
- bcrypt
- validator
- Helmet.js
- Winston Logger

---

## Week 1 - Security Assessment

The following security testing activities were performed:

- OWASP ZAP automated scanning
- Cross-Site Scripting (XSS) testing
- SQL Injection testing
- Browser Developer Tools testing

### Vulnerabilities Identified

- Cross-Site Scripting (XSS)
- SQL Injection
- Missing Security Headers
- Security Misconfigurations

---

## Week 2 - Security Improvements

Security measures implemented:

- Input validation using validator
- Password hashing using bcrypt
- JWT-based authentication
- Secure HTTP headers using Helmet.js
- Logging using Winston

---

## Week 3 - Advanced Security

Activities performed:

- Basic penetration testing
- Nmap localhost scanning
- Logging verification
- Security checklist preparation
